#  Online shop app | React + TypeScript

A React application for browsing products, add them to favorites or add them to cart.

## Features

- Search product by phrase
- Search by category product
- Add and remove from favorite list
- Add and remove from cart
- View detailed product page
- Authentication:
  - Sign Up (stored in localStorage)
  - Sign In
  - Logout
- Statistics available (admin)
- echo chat with employees

## Tech Stack

- React
- TypeScript
- Tanstack Router
- Tanstack Query
- Taiwind libray
- DummyJSON API

## How to run application

1. Clone repository using
   git clone https://github.com/igap-git/shop-app-react.git
2. npm install
3. npm start
